# ebabel-skybox
Setup a skybox mesh and add it to the scene.

## Instructions for Module Usage in your project/game

### Install
```
npm i --save ebabel-skybox
```

### Usage
```
const skybox = require('ebabel-skybox');

skybox({ THREE, scene });
```
